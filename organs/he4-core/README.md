# He-4 Core — Cryogenic Thermal Buffer

**Status: DESIGNED**

## Function

The circulatory system of the Ghost Shell. Liquid Helium-4 phase-change channels that maintain quasi-isothermal conditions at the MTR core and transport heat outward through the PRF frame.

## Key Properties

- **Phase-change buffer**: Absorbs heat spikes via liquid-gas transition (high latent heat)
- **Capillary routing**: Channels integrated into PRF structure (no external plumbing)
- **Superfluid potential**: He-4 below lambda point enables extremely efficient heat transport
- **Active control**: Microvalves and cryocoolers regulate flow in response to thermal sensors

## Integration

- Channels run beneath MTR track for direct quench protection
- Heat carried through PRF thermal highway to Electrodermus for radiation
- Umbilical loops connect through PRF structure to MTR mounts
- Provides the "cryogenic metabolism" — homeostatic temperature regulation
