# He-4 Umbilicals — Service Ports

**Status: DESIGNED**

## Function

Cryogenic and telemetry service ports integrated into the PRF structure. The umbilical cord of the Ghost Shell — connecting internal organs to external service, diagnostics, and supply.

## Key Properties

- **Cryo supply**: He-4 fill/vent lines for cryogenic system servicing
- **Telemetry**: Data ports for organ state monitoring and firmware updates
- **Integrated routing**: Built into PRF truss members (no external plumbing)
- **Redundant paths**: Multiple umbilical loops ensure single-point failure tolerance
