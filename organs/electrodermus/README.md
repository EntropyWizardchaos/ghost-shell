# Electrodermus — Interface Skin

**Status: DESIGNED**

## Function

The outer interface of the Ghost Shell. Handles power harvesting, radiation control, and tactile/optical sensing. The skin that breathes.

## Key Properties

- **Photonic fur**: Radiating structures that increase effective surface area by 30-40%
- **DLC sheath**: Diamond-like carbon protective coating
- **Heater lanes**: 100 Ohm at 12V = 1.44W per lane; 20s pulse = 28.8J
- **EDLC storage**: 2-4F supercapacitors per quadrant (1F/12V stores 72J)
- **PV harvesting**: Integrated photovoltaic surfaces for ambient energy capture

## Integration

- Receives waste heat from PRF thermal highway and radiates to environment
- Large surface area (potentially augmented with fins) maximizes thermal dissipation
- Tactile/optical sensing feeds environmental data to Cognitive Lattice
- Active thermal zones controllable per-quadrant via EDLC-powered heater lanes
